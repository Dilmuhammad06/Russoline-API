
# Russoline API

This is a Russoline API project.
Russoline is a social media web site.

To use install all from list requirements.txt.

## Running

To run on Windows machine

```bash
  python manage.py runserver
```


To run on Linux or MacOS

```bash
  python3 manage.py runserver
```


## Features

- User auth
- Content posting
- Follow system
- Like system
- Comment system
